# MineSweeper Run

MineSweeper Run是一款基于Windows控制台的扫雷游戏程序，Window版本可在图形界面窗口进行游戏。

**控制台部分特性：**

（1）默认生成可解地图，无需面对死亡二选一的局面

（2）地图求解器支持简单判断、逻辑推理、枚举判断、雷率计算、剩余雷数判断等完善的扫雷逻辑，且运行高效

（3）在控制台的基于鼠标和键盘混动的操作界面，以及指令、WASD、鼠标三种操作混动的游戏界面

（4）完整的扫雷难度支持，地图从1\*1到256\*384，雷数从0到满（大地图建议使用Window版本）

（5）生成缓存文件，自动保存用户设置、上一次地图和历史记录，胜利时显示打破纪录

（6）可对外部输入的地图进行求解、游戏时按!/Tab对实时求解、连续生成地图并不断求解（压榨CPU）

（7）控制台可切换深浅模式、设置自选光标、设置是否实时显示用时和3BV，界面个性化

**图形界面窗口部分特性：**

（1）Ctrl+滚轮调整显示大小

（2）Ctrl+左键拖动调整地图位置

（3）点击笑脸按钮或按N重新生成地图
